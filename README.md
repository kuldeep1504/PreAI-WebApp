AI-Powered Interview Preparation System

📌 Project Description

The AI-Powered Interview Preparation System is a modern MERN stack web application designed to help students and job seekers prepare for technical, HR, and coding interviews using advanced AI technologies. The platform provides personalized interview preparation, AI-generated questions, live mock interviews, resume analysis, coding practice, voice interaction, and performance tracking.

🚀 Features:-

🔐 Authentication System

- User Registration & Login
- JWT Authentication
- Password Encryption using bcrypt
- Forgot Password & Email Verification
- Role-Based Access (Student/Admin)

👤 User Profile

- Upload Profile Photo
- Resume Upload
- Add Skills & Experience
- Select Target Companies
- Interview Category Selection

🤖 AI Features

🎤 Voice & Video Interview

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/AI-Interview-System.git
```

## 2️⃣ Install Frontend Dependencies

```bash
cd client
npm install
```

## 3️⃣ Install Backend Dependencies

```bash
cd ../server
npm install
```

## 4️⃣ Setup Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key

OPENAI_API_KEY=your_openai_key
GEMINI_API_KEY=your_gemini_key

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```

# ▶️ Run Project

## Start Backend

```bash
cd server
npm run dev
```

## Start Frontend

```bash
cd client
npm run dev
```
